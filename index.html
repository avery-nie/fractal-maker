fractal maker


Instructions: CLICK the square tiles on the left to change the pattern.
UP & DOWN arrow keys change the number of iterations of the fractal.
LEFT & RIGHT arrow keys change the grid size.


Note: this code is inefficient and this website may cook your device.

I initially made this for a high school calculus project in 2022. You can view the original OpenProcessing sketch online here. A runnable Processing file is available in the repo for this site.
